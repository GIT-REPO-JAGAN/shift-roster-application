Rotation Model
Weekly  /  Biweekly  /  Monthly  /  Static
  • If all groups have a single shift (e.g. [( N ), ( A ), ( G,G ), ( E )]) → Cross-group rotation using a shared pool [N, A, G, G, E].
  • If any group has multiple shifts (e.g. [( N,M,A ), ( M,N,A )]) → Intra-group rotation within each group.
